# Code of Conduct

This project is built in public and welcomes contributors from anywhere. A few ground rules keep that collaboration healthy.

## Expected behavior

- Be respectful. Assume good intent.
- Stay on topic. This is a technical project; keep discussions technical.
- Critique code, not people. Focus feedback on the work.
- Help newcomers. Everyone was new once.

## Unacceptable behavior

- Harassment, personal attacks, or discrimination of any kind.
- Publishing someone else's private information without consent.
- Spam, off-topic promotion, or political flamewars.
- Bad-faith arguments that waste contributors' time.

## Enforcement

The project lead resolves disputes. Report issues privately to **hi@iamdejan.com**. Serious or repeated violations result in a ban from the repo and associated channels.

## Scope

This applies to GitHub issues, pull requests, discussions, and any official Mesh communication channel.
